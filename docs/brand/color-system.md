# Color System

## Canonical Colors

| Token | Hex | Use |
| --- | --- | --- |
| Deep navy | `#0B1F33` | Primary identity color, dark backgrounds, dark text |
| Signal blue | `#2F80ED` | Primary signal accent and arcs |
| Light signal blue | `#4EA1FF` | Signal arcs on dark backgrounds |
| Audit amber | `#F2A900` | Controlled point accent, audit emphasis |
| White | `#FFFFFF` | Light surfaces and reversed text |
| Ink | `#0A0A0A` | High-contrast text where navy is not appropriate |
| Light gray | `#E6E8EC` | Borders, quiet fills, and documentation surfaces |

## Guidance

Deep navy should carry the brand weight. Signal blue should be used deliberately for intelligence signals, links, and key accents. Audit amber is a small emphasis color and should not dominate layouts.

Avoid gradients, glow effects, cyberpunk palettes, purple AI palettes, and decorative color treatments that weaken the controlled enterprise tone.
