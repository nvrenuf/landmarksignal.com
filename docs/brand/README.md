# LandmarkSignal Brand Kit

This directory documents the v1 LandmarkSignal Signal Beacon identity system.

## Source Of Truth

The canonical brand assets are SVG files in `public/brand`. Do not replace them with generated raster artwork or use raster exports as source files.

## Brand Idea

LandmarkSignal uses the Signal Beacon mark: a stable vertical landmark point with controlled signal arcs above it. The mark combines three ideas:

- Stable reference point
- Intelligence signal
- Governed control

The tone is enterprise, controlled, secure, and serious.
